This directory contains the preliminary data files under development
for Version 14.0.0 of the Unicode Standard.

The "ucd" subdirectory contains the Unicode
Character Database data files.

The "charts" subdirectory WILL contain an archival set of
pdf code charts corresponding exactly to Version 14.0.0.

The "ucdxml" subdirectory WILL contain the XML version of
the Unicode Character Database.

Zipped versions of UCD data files for Version 14.0.0 WILL BE
posted in:

http://www.unicode.org/Public/zipped/14.0.0/
